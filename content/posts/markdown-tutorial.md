# Markdown 入门教程

Markdown 是一种轻量级标记语言，让你使用易读易写的纯文本格式编写文档，然后转换成结构化的 HTML 文档。

<!--more-->

## 1. 标题

使用 `#` 符号表示标题，数量越多级别越低：

```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
```

## 2. 段落和换行

段落之间用空行分隔：

```markdown
这是第一段。

这是第二段。
```

## 3. 强调

- **粗体**：用两个星号 `**粗体**` 或两个下划线 `__粗体__`
- *斜体*：用一个星号 `*斜体*` 或一个下划线 `_斜体_`
- ~~删除线~~：用两个波浪线 `~~删除线~~`

## 4. 列表

### 无序列表

```markdown
- 项目一
- 项目二
- 项目三
```

效果：
- 项目一
- 项目二
- 项目三

### 有序列表

```markdown
1. 第一步
2. 第二步
3. 第三步
```

效果：
1. 第一步
2. 第二步
3. 第三步

## 5. 链接和图片

### 链接

```markdown
[链接文字](https://example.com)
```

效果：[链接文字](https://example.com)

### 图片

```markdown
![图片描述](图片地址)
```

## 6. 引用

```markdown
> 这是一段引用文字。
> 可以有多行。
```

效果：
> 这是一段引用文字。
> 可以有多行。

## 7. 代码

### 行内代码

```markdown
使用 `print()` 函数输出内容。
```

效果：使用 `print()` 函数输出内容。

### 代码块

使用三个反引号包裹代码，并可指定语言：

````markdown
```python
def hello():
    print("Hello, World!")
```
````

效果：
```python
def hello():
    print("Hello, World!")
```

## 8. 分隔线

使用三个或更多的星号、减号或下划线：

```markdown
---

***

___
```

效果：

---

## 9. 表格

```markdown
| 姓名 | 年龄 | 城市 |
|------|------|------|
| 张三 | 25   | 北京 |
| 李四 | 30   | 上海 |
| 王五 | 28   | 广州 |
```

效果：

| 姓名 | 年龄 | 城市 |
|------|------|------|
| 张三 | 25   | 北京 |
| 李四 | 30   | 上海 |
| 王五 | 28   | 广州 |

## 10. 任务列表

```markdown
- [x] 已完成任务
- [ ] 未完成任务
- [ ] 另一个未完成任务
```

效果：
- [x] 已完成任务
- [ ] 未完成任务
- [ ] 另一个未完成任务

## 总结

Markdown 语法简洁明了，适合写作、文档编写和技术博客。掌握这些基础语法，你就可以开始用 Markdown 编写精美的文档了！
